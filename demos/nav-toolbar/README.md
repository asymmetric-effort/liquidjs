# Toolbar

Interactive demo of the Toolbar horizontal button group component.

## Features

- Button items with icons
- Toggle active state
- Separators and spacers
- Size variants (sm, md, lg)
- Flat and raised variants
- Disabled buttons

## Quick Start

```bash
cd ui && npm install && npm run dev
```

## How to Run E2E Tests

```bash
npx playwright test
```
