# Form TimePicker Demo

Interactive demo of the TimePicker form component - time selection with hour/minute spinners.

## Features

- useState, useCallback, createElement hooks
- Form components integration
- Time selection with hour/minute spinners
- 12-hour and 24-hour format support
- Configurable minute step intervals
- Disabled and error states

## Quick Start

```bash
cd ui && npm install && npm run dev
```

The app will be available at `http://localhost:5173`.

## E2E Tests

Install Playwright and run:

```bash
npx playwright install --with-deps
npx playwright test
```
