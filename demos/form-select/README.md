# Form Select Demo

Interactive demo of the Select form component - dropdown with search, multi-select, clearable.

## Features

- useState, useCallback, createElement hooks
- Form components integration
- Single select dropdown with search
- Multi-select with tag display
- Clearable selection
- Disabled and error states

## Quick Start

```bash
cd ui && npm install && npm run dev
```

The app will be available at `http://localhost:5173`.

## E2E Tests

Install Playwright and run:

```bash
npx playwright install --with-deps
npx playwright test
```
