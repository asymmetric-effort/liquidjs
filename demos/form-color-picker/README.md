# Form Color Picker Demo

Interactive demo of the ColorPicker form component - color swatch grid with hex input.

## Features

- useState, useCallback, createElement hooks
- Form components integration
- Color selection with swatch grid and hex input
- Custom color presets
- Option to hide input field
- Disabled state support

## Quick Start

```bash
cd ui && npm install && npm run dev
```

The app will be available at `http://localhost:5173`.

## E2E Tests

Install Playwright and run:

```bash
npx playwright install --with-deps
npx playwright test
```
