# Todo App Demo

A standalone SpecifyJS SPA demonstrating core framework features.

## Features Demonstrated

- `useState` — Todo list state, filter state, dark mode toggle
- `useRef` — Input focus management, auto-incrementing ID
- `useEffect` — LocalStorage persistence, document title updates
- `useCallback` — Stable event handlers
- `useMemo` — Filtered list computation, remaining count
- Keyed list reconciliation
- Conditional rendering
- Event handling (click, keydown, change)

## Run

```bash
docker compose up --build
```

Open http://localhost:3000
