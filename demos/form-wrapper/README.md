# Form FormFieldWrapper Demo

Interactive demo of the FormFieldWrapper form component - label, helpText, error, required, disabled styling.

## Features

- useState, useCallback, createElement hooks
- Form components integration
- Base container for all form field components
- Label, help text, and error message display
- Required field indicator
- Disabled state styling

## Quick Start

```bash
cd ui && npm install && npm run dev
```

The app will be available at `http://localhost:5173`.

## E2E Tests

Install Playwright and run:

```bash
npx playwright install --with-deps
npx playwright test
```
