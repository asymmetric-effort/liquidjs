# Form DatePicker Demo

Interactive demo of the DatePicker form component - calendar dropdown with date constraints.

## Features

- useState, useCallback, createElement hooks
- Form components integration
- Calendar dropdown with month navigation
- Min/max date constraints
- Custom date format support
- Disabled and error states

## Quick Start

```bash
cd ui && npm install && npm run dev
```

The app will be available at `http://localhost:5173`.

## E2E Tests

Install Playwright and run:

```bash
npx playwright install --with-deps
npx playwright test
```
