# Form MultilineField Demo

Interactive demo of the MultilineField form component - textarea, auto-resize, character count.

## Features

- useState, useCallback, createElement hooks
- Form components integration
- Multi-line textarea with character count
- Auto-resize with configurable min/max height
- Disabled and error states
- Required field support

## Quick Start

```bash
cd ui && npm install && npm run dev
```

The app will be available at `http://localhost:5173`.

## E2E Tests

Install Playwright and run:

```bash
npx playwright install --with-deps
npx playwright test
```
