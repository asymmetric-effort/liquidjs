# Splitter

Interactive demo of the Splitter resizable split pane with draggable divider, direction toggle, and size constraints.

## Features

- Horizontal and vertical split directions
- Draggable divider with mouse events
- Configurable initial split percentage
- Min size constraint
- Adjustable divider size
- useState for interactive controls

## Quick Start

```bash
cd ui && npm install && npm run dev
```

## How to Run E2E Tests

```bash
npx playwright test
```
