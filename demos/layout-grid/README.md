# Grid

Interactive demo of the Grid CSS Grid layout component with configurable columns, gap, auto-fit, and alignment.

## Features

- Grid component with numeric/string columns
- GridItem with span and area placement
- Auto-fit responsive columns with minColWidth
- Configurable gap and alignment
- useState for interactive controls

## Quick Start

```bash
cd ui && npm install && npm run dev
```

## How to Run E2E Tests

```bash
npx playwright test
```
