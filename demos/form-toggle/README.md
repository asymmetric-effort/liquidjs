# Form Toggle Demo

Interactive demo of the Toggle form component - toggle switch with sliding animation.

## Features

- useState, useCallback, createElement hooks
- Form components integration
- Toggle switch with sliding animation
- Size variants (sm, md, lg)
- Label position (left, right)
- Custom on-color support

## Quick Start

```bash
cd ui && npm install && npm run dev
```

The app will be available at `http://localhost:5173`.

## E2E Tests

Install Playwright and run:

```bash
npx playwright install --with-deps
npx playwright test
```
