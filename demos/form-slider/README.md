# Form Slider Demo

Interactive demo of the Slider form component - range slider with marks and dual handles.

## Features

- useState, useCallback, createElement hooks
- Form components integration
- Single value slider with step control
- Range slider with dual handles
- Tick marks and custom labels
- Show/hide value label option

## Quick Start

```bash
cd ui && npm install && npm run dev
```

The app will be available at `http://localhost:5173`.

## E2E Tests

Install Playwright and run:

```bash
npx playwright install --with-deps
npx playwright test
```
