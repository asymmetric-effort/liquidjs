# FlexContainer

Interactive demo of the FlexContainer component with configurable direction, wrap, gap, alignment, and FlexItem controls.

## Features

- FlexContainer with direction and wrap
- FlexItem with flex, grow, shrink, basis
- Configurable gap, alignItems, justifyContent
- Inline flex toggle
- useState for interactive controls

## Quick Start

```bash
cd ui && npm install && npm run dev
```

## How to Run E2E Tests

```bash
npx playwright test
```
