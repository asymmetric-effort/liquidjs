# Menubar

Interactive demo of the Menubar application menu bar component.

## Features

- Top-level menu triggers
- Dropdown menu panels
- Dividers between item groups
- Keyboard shortcut labels
- Disabled menu items
- Close on Escape

## Quick Start

```bash
cd ui && npm install && npm run dev
```

## How to Run E2E Tests

```bash
npx playwright test
```
