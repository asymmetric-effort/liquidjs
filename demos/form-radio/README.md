# Form RadioGroup Demo

Interactive demo of the RadioGroup form component - radio button group with keyboard navigation.

## Features

- useState, useCallback, createElement hooks
- Form components integration
- Radio button group with keyboard navigation
- Vertical and horizontal layout directions
- Individual option disabled state
- Error state display

## Quick Start

```bash
cd ui && npm install && npm run dev
```

The app will be available at `http://localhost:5173`.

## E2E Tests

Install Playwright and run:

```bash
npx playwright install --with-deps
npx playwright test
```
