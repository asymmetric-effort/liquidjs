# Stepper

Interactive demo of the Stepper step wizard indicator component.

## Features

- Step progression with prev/next
- Clickable steps
- Horizontal and vertical orientation
- Circle and dot variants
- Completed/active/pending states
- Step descriptions

## Quick Start

```bash
cd ui && npm install && npm run dev
```

## How to Run E2E Tests

```bash
npx playwright test
```
