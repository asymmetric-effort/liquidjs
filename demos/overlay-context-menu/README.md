# Context Menu

Right-click context menu demo with nested submenus, keyboard navigation, and disabled items.

## Features

- useState
- useCallback
- Right-click trigger
- Nested submenus
- Keyboard navigation (arrows, enter, escape)
- Disabled items
- Divider separators
- Click outside close

## Quick Start

```bash
cd ui && npm install && npm run dev
```

## How to Run E2E Tests

```bash
npx playwright test
```
