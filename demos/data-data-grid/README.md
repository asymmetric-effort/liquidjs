# DataGrid

Interactive DataGrid demo with sorting, filtering, pagination, and row selection.

## Features

- useState
- useCallback
- useMemo
- Sorting
- Filtering
- Pagination
- Row selection
- Striped rows
- Sticky header

## Quick Start

```bash
cd ui && npm install && npm run dev
```

## How to Run E2E Tests

```bash
npx playwright test
```
