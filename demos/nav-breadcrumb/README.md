# Breadcrumb

Interactive demo of the Breadcrumb trail navigation component.

## Features

- Breadcrumb trail with links
- Collapsible middle items with maxItems
- Custom separators
- Size variants (sm, md, lg)
- Interactive path navigation

## Quick Start

```bash
cd ui && npm install && npm run dev
```

## How to Run E2E Tests

```bash
npx playwright test
```
