# VirtualScroll

VirtualScroll demo rendering 10,000+ items efficiently with configurable height and overscan.

## Features

- useState
- useCallback
- useRef
- Virtual scrolling
- Overscan buffer
- Large datasets
- Scroll events

## Quick Start

```bash
cd ui && npm install && npm run dev
```

## How to Run E2E Tests

```bash
npx playwright test
```
