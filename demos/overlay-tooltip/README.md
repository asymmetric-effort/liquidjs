# Tooltip

Lightweight hover tooltip demo with placement variants, delay controls, and max width.

## Features

- useState
- Hover and focus triggers
- Placement variants (top, bottom, left, right)
- Configurable delay
- Max width text wrapping
- Arrow indicator

## Quick Start

```bash
cd ui && npm install && npm run dev
```

## How to Run E2E Tests

```bash
npx playwright test
```
