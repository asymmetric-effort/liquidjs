# Panel

Interactive demo of the Panel component with collapsible body, icon, header slots, and configurable border/shadow.

## Features

- Panel with title and icon
- Collapsible with animated max-height transition
- Header right slot
- Default collapsed state
- Configurable border and shadow
- useState for interactive controls

## Quick Start

```bash
cd ui && npm install && npm run dev
```

## How to Run E2E Tests

```bash
npx playwright test
```
