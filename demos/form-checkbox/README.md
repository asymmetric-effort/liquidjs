# Form Checkbox Demo

Interactive demo of the Checkbox form component - custom checkbox with indeterminate state.

## Features

- useState, useCallback, createElement hooks
- Form components integration
- Checkbox with label, disabled, error, and indeterminate states
- Size variants (sm, md, lg)
- Live preview with interactive controls

## Quick Start

```bash
cd ui && npm install && npm run dev
```

The app will be available at `http://localhost:5173`.

## E2E Tests

Install Playwright and run:

```bash
npx playwright install --with-deps
npx playwright test
```
