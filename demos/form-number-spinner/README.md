# Form NumberSpinner Demo

Interactive demo of the NumberSpinner form component - numeric input with increment/decrement.

## Features

- useState, useCallback, createElement hooks
- Form components integration
- Numeric input with increment/decrement buttons
- Keyboard support for value changes
- Prefix and suffix display
- Configurable min, max, and step values

## Quick Start

```bash
cd ui && npm install && npm run dev
```

The app will be available at `http://localhost:5173`.

## E2E Tests

Install Playwright and run:

```bash
npx playwright install --with-deps
npx playwright test
```
