# Popover

Positioned popover demo with placement variants, controlled mode, and arrow support.

## Features

- useState
- useCallback
- Controlled and uncontrolled modes
- Placement variants (top, bottom, left, right)
- Arrow indicator
- Click outside close

## Quick Start

```bash
cd ui && npm install && npm run dev
```

## How to Run E2E Tests

```bash
npx playwright test
```
