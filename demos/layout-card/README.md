# Card

Interactive demo of the Card component with title, subtitle, image, footer, and configurable shadow/border/hover.

## Features

- Card with header, body, and footer slots
- Top image with alt text
- Header action slot
- Configurable shadow, border, padding, radius
- Hoverable effect
- useState for interactive controls

## Quick Start

```bash
cd ui && npm install && npm run dev
```

## How to Run E2E Tests

```bash
npx playwright test
```
