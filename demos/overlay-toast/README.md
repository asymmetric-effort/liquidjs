# Toast

Toast notification system demo with type variants, actions, persistent toasts, and dismiss controls.

## Features

- useState
- useCallback
- useRef
- createToaster factory
- Toast types (info, success, warning, error)
- Action buttons
- Persistent toasts
- Dismiss all
- Auto-dismiss timer

## Quick Start

```bash
cd ui && npm install && npm run dev
```

## How to Run E2E Tests

```bash
npx playwright test
```
