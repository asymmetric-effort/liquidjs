# ScrollContainer

Interactive demo of the ScrollContainer with configurable direction, scrollbar visibility, edge shadows, and padding.

## Features

- Vertical, horizontal, and both scroll directions
- Scrollbar visibility modes (auto, always, hover, never)
- Inset edge shadows on scroll
- Configurable maxHeight, maxWidth, padding
- Dynamic item count
- useState for interactive controls

## Quick Start

```bash
cd ui && npm install && npm run dev
```

## How to Run E2E Tests

```bash
npx playwright test
```
