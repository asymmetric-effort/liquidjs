# Form TextEditor Demo

Interactive demo of the TextEditor form component - WYSIWYG editor with formatting toolbar.

## Features

- useState, useCallback, createElement hooks
- Form components integration
- WYSIWYG rich text editor with formatting toolbar
- Configurable toolbar buttons
- Read-only mode
- Disabled and error states

## Quick Start

```bash
cd ui && npm install && npm run dev
```

The app will be available at `http://localhost:5173`.

## E2E Tests

Install Playwright and run:

```bash
npx playwright install --with-deps
npx playwright test
```
