# Form FileUpload Demo

Interactive demo of the FileUpload form component - drag-and-drop file upload with validation.

## Features

- useState, useCallback, createElement hooks
- Form components integration
- Drag-and-drop file upload zone
- File type filtering with accept attribute
- Max file size validation
- Single and multiple file upload modes

## Quick Start

```bash
cd ui && npm install && npm run dev
```

The app will be available at `http://localhost:5173`.

## E2E Tests

Install Playwright and run:

```bash
npx playwright install --with-deps
npx playwright test
```
