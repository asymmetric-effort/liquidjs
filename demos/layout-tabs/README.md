# Tabs

Interactive demo of the Tabs component with position, variant, disabled tabs, and keyboard navigation.

## Features

- Tab list with line, card, and pill variants
- Position top, bottom, left, right
- Disabled tab support
- Keyboard navigation (arrow keys, Home, End)
- ARIA tablist/tab/tabpanel roles
- Controlled and uncontrolled active tab
- useState for interactive controls

## Quick Start

```bash
cd ui && npm install && npm run dev
```

## How to Run E2E Tests

```bash
npx playwright test
```
