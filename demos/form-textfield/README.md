# Form TextField Demo

Interactive demo of the TextField form component - single-line input, prefix/suffix, validation.

## Features

- useState, useCallback, createElement hooks
- Form components integration
- Single-line text input with label and help text
- Password field type
- Prefix and suffix element support
- Disabled, error, and required states

## Quick Start

```bash
cd ui && npm install && npm run dev
```

The app will be available at `http://localhost:5173`.

## E2E Tests

Install Playwright and run:

```bash
npx playwright install --with-deps
npx playwright test
```
