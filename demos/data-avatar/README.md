# Avatar

Avatar demo with image, initials fallback, status indicators, sizes, and shapes.

## Features

- useState
- Image avatars
- Initials fallback
- Status indicators
- Size presets
- Custom sizes
- Circle/square shapes
- Custom colors

## Quick Start

```bash
cd ui && npm install && npm run dev
```

## How to Run E2E Tests

```bash
npx playwright test
```
