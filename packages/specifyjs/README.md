# specifyjs

This is an alias package for [`@asymmetric-effort/specifyjs`](https://www.npmjs.com/package/@asymmetric-effort/specifyjs).

Both packages are identical. Use whichever you prefer:

```bash
npm install specifyjs
# or
npm install @asymmetric-effort/specifyjs
```

See [specifyjs.asymmetric-effort.com](https://specifyjs.asymmetric-effort.com) for full documentation.
